# durhack2024-adventure-game

## To run the app locally

Install Python libraries:
```bash
pip install -r requirements
```

Run flask server:
```bash
python main.py
```

Install npm dependencies:
```bash
cd App/web-ui/
npm install
```

Run React app:
```bash
npm run dev
```

Access the game on http://localhost:5173/the-broken-crown/
